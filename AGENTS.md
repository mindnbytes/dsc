# AGENTS.md

Read this file before working in this repository.

## Project Goal

`dsc` is a minimal educational C project for learning C engineering through implementing common data structures.

The priority is clarity, correctness, and learning value. Prefer simple, readable C over clever abstractions or production-heavy infrastructure.

## Agent Guidance

- Read the existing code and tests before suggesting or changing anything.
- Keep changes small and directly related to the requested task.
- Preserve the educational style of the project.
- Prefer explicit ownership rules and simple APIs.
- Add or update focused tests when behavior changes.
- Avoid unrelated refactors, dependencies, or tooling unless specifically requested.
